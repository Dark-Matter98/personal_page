# Prashant Dhakal Portfolio

Static single-page CV and portfolio site. There is no build step.

## Local preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Update checklist

- Replace `headshot.jpg` if you want to update the web headshot.
- Replace `run.jpeg`, `run2.jpg`, and `lift.JPG` if you want to update the Hobbies section photos.
- Update links for LinkedIn, GitHub, ORCID, or personal project demos if you want them added.
- Replace the PDFs when your resume/CV changes:
  - `Prashant_Dhakal_Industry_Resume.pdf`
  - `Prashant_Dhakal_Academic_CV.pdf`
- Edit rotating hero titles in `script.js`.
- Edit profile content, projects, publications, and contact details in `index.html`.

## GitHub Pages deployment

1. Create a GitHub repository, for example `prashant-portfolio`.
2. Upload the contents of this `portfolio` folder to the repository root.
3. In GitHub, go to `Settings -> Pages`.
4. Choose `Deploy from a branch`, select `main`, and select `/root`.
5. Your site will publish at `https://YOUR_USERNAME.github.io/prashant-portfolio/`.

For a root personal website, name the repository `YOUR_USERNAME.github.io`; then the URL will be `https://YOUR_USERNAME.github.io/`.
